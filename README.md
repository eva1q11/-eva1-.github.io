
# EGOR KREED · FANPAGE (RU)

Многостраничный шаблон фан-сайта (GitHub Pages). Русский интерфейс, аккуратная "звёздная" тема.

## Страницы
- `index.html` — Главная (герой-блок, карточки)
- `bio.html` — Биография
- `discography.html` — Дискография (карточки альбомов/синглов)
- `timeline.html` — Хронология (события по годам)
- `gallery.html` — Галерея (фото + YouTube iframe)
- `contacts.html` — Контакты (демо-форма)

## Деплой (GitHub Pages)
1. Создайте репозиторий, загрузите файлы.
2. Settings → Pages → Deploy from a branch → `main` / root.
3. Откройте выданный адрес.

## Как заменить контент
- Тексты правьте прямо в HTML.
- Картинки кладите в `/assets` и меняйте пути (в `gallery.html` и `discography.html`).
- Цвета темы в `style.css` (CSS-переменные в начале).
- Вставьте ID клипа YouTube в `gallery.html` (замените `VIDEO_ID`).

## Рекомендации по контенту
- Указывайте источники фото и соблюдайте авторские права.
- Проверяйте отображение на телефоне.
- Для форм подключите Formspree/Netlify Forms (сейчас alert-демо).
